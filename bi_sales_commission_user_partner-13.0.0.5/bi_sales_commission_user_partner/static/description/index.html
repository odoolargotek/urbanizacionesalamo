Sales Commission for Users/Salesperson and Partners Odoo Apps

This Odoo apps use to calculate sales commission from the sales order/account invoice and Account Payment.It also calculate commission based on product, product category or sales team based on configuration. After installing this odoo apps once commission is calculated and generated you can see commission worksheet and also able to generate Sales commission worksheet PDF Report. You can also able to make vendor bills/supplier invoice easily from the commission lines for sales team /sales person also for external partners.

Features

Commission Level.

Setup different commission level as Manager, Salesman etc.

Dynamic Commission Calculation.

Dynamic Commission Calculation based on percentage defined in product, product category, partner or sales team.

Commission Payment Configuration.

Configure when to pay commission such as while sale order confirmation, while invoice validation, while payment confirmation.

Commission Worksheet Report.

You can print Commission Worksheet Report available in PDF.

Generate Commission Worksheet from Commission Line.

User have option to select commission lines and generate commission worksheet from commission lines.

Commission Lines SQL and Pivot Tables and PDF Report.

User can able to generate Pivot SQL report as well as PDF report for generated commissions.

Commission Level

Go to Sales--> Configuration --> Commission Level. Here you can create or edit different Commission Level such as Manager, Broker etc.

Product Configuration

Setup Commission Levels and commission percentage.

Configuration for Sales Team.

Configuration for Product Category.

Configuration for Partner

Commission Settings

Go to Sales --> Settings. Find 'Commission Configuration' group. In 'Pay Commission Based On' field select an option as when you want to pay the commission. In 'Commission Calculation Based On' field, select an option as based upon what you want commission gets calculated.

Commission Based on Sales Order and Commission Calculation based on Product

Create Quotation with commission levels and assign User/Partner for each level in Sales Commission User Settings tab. Then Confirm the Sale Order to create commissions.

Commission Lines

If you have configured Pay Commission based on as 'Sales Order', and Commission Calculation based on as Product, then While confirming the Sale Order Commission Lines will be created.

User can see commission line pivot view.

User can see commission line graph view.

Commission Settings based on Invoice

Go to Sales --> Settings. Find 'Commission Configuration' group. In 'Pay Commission Based On' field select 'Commission based on Invoice'. In 'Commission Calculation Based On' field, select an option as based upon what you want commission gets calculated.

Commission Based on Invoice Validation

Create Invoice with commission levels and assign sales team which is configured with commission. Then Validate the Invoice to create commissions.

Commission Lines

If you have configured Pay Commission based on as 'Invoice', and Commission Calculation based on as Sales Team, then While validating the Invoice, Commission Lines will be created.

User can see commission line pivot view.

User can see commission line graph view.

Commission Settings based on Payment

Go to Sales --> Settings. Find 'Commission Configuration' group. In 'Pay Commission Based On' field select 'Commission based on Payment'. In 'Commission Calculation Based On' field, select an option as based upon what you want commission gets calculated.

Commission Based on Payment Validation

Create Payment with commission levels and assign User/Partner whom is configure with commission. Then Validate the Payment to create commissions.

Commission Lines

If you have configured Pay Commission based on as 'Payment', and Commission Calculation based on as Partner, then While validating the Payment, Commission Lines will be created.

User can see commission line pivot view.

User can see commission line graph view.

Commission Worksheet

Commission Worksheet will be automatically created for a month. You can create Invoice from the monthly commission worksheet for a particular vendor.

Commission Worksheet Invoice

Invoice will be created with the total commission worksheet amount and Customer.

Commission Worksheet Report

You can Download PDF Report from Commission Worksheet form
